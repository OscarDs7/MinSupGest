Título de la Aplicación Móvil
-----------------------
MinSupGest

Propósito
----------
Gestionar inventarios, empleados y generar estadísticas y recomendaciones para la buena toma de decisiones de las microempresas.

Tecnología usada
----------------
- Android Studio
- Lenguaje Kotlin
- Firebase
- Git / GitHub
